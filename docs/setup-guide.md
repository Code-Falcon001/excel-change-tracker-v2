# 🚀 Setup Guide

## Step 1: Download Manifest
- Download **`manifest_simple.xml`** from this repo

## Step 2: Upload to Excel Online
1. Open your workbook in **Excel Online**
2. **Home** → **Add-ins** → **Upload My Add-in**
3. Select **`manifest_simple.xml`** → Upload

## Step 3: First-Time Setup
1. Task pane opens → sheet is **🔒 locked**
2. Enter your name → click **Save & Start Tracking**
3. Sheet **🔓 unlocks** → tracking begins

## Step 4: Verify
1. Edit any cell in "STD format"
2. Check task pane log + "changelog" sheet

## Returning Users
- Name saved in browser → auto-starts next time

## Troubleshooting

| Problem | Solution |
|---|---|
| "Manifest not valid" | Use `manifest_simple.xml` (not full) |
| Wrong content in pane | Clear cache (Ctrl+Shift+Del) |
| "Sheet not found" | Sheet must be named "STD format" |
| Changes not logging | Keep task pane open |
